# Unity Module

contains code for importing eyetracking data from a .csv file.

format:

| tobii_timestamp | GazeX | GazeY | ... |
|-----------------|-------|-------|-----|
| int             | float | float | ... |

or

| timestamp | X     | Y     | ... |
|-----------|-------|-------|-----|
| int       | float | float | ... |

> this module is mostly for my personal use, you can use it as an example for how to use the "Heatmap" module
